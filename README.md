## The Problem

Public health data is everywhere.

But it is scattered, isolated, and often misleading when viewed alone.

- Nutrition is analyzed separately from disease  
- Demographics are treated as background variables  
- Insights remain shallow because systems are never seen as a whole  

> **Decisions end up being informed — but not complete.**

---

## The Idea

HealthScape reframes the lens.

It does not look at indicators individually.  
It looks at how they interact.

```
Isolated Metrics  →  Connected Systems
```

Clarity over complexity.

---

## Key Features

- Unified multi-indicator health view  
- Cross-dimensional analysis (nutrition × disease × demographics)  
- Clean, interpretable visual outputs  
- Region-wise comparison and clustering  
- Reproducible and structured workflow
